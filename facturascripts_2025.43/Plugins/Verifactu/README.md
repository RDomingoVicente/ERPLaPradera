# Verifactu
Permite la integración con Verifactu con FacturaScripts para la emisión de facturas electrónicas.
- https://facturascripts.com/plugins/verifactu

## Nombre de carpeta
Como con todos los plugins, la carpeta se debe llamar igual que el plugin. En este caso **Verifactu**.
